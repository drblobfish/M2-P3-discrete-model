# Cellular Potts Model


```
make UI
```

compiles the project. You might want to change the compiler to whatever c++ compiler is available on your machine. No external dependencies are needed except the c++ standard library. `raylib` is included in the `extern` folder.

run with
```
./UI [param_file.txt]
```

examples of parameter files are available in `param`.


This project is produced as part of the course "Discrete Models in Biology" of Sorbonne University, under supervision by Sophie Hecht and Diane Peurichard.
